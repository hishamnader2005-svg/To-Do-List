# To-Do List (Java)

A simple To-Do List application built in Java. This project helps practice fundamental Java concepts such as object-oriented programming, collections, user input handling, and basic application logic.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Simple and easy-to-use interface
- Written entirely in Java

## Technologies Used

- Java (JDK 8 or higher recommended)
- Java Standard Library

## How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/todo-list-java.git

2. Navigate to the project directory
   cd todo-list-java

3. Compile the java files
   javac *.java

4. Run the Applications
   java control
